最新发现👀👀👀👀Prodia 好像你调用了api，次数也还是1000次 好像可以一直白嫖

昨天发现了Prodia 有免费的Stable Diffusion 的API提供，我的想法是能白嫖绝不花钱💩白嫖Stable Diffusion需要注册两个网站

💡Prodia 一个账号能白嫖1000次，用完可换一个邮箱继续白嫖

  • huggingface
  • Prodia
  1. 第一步打开Prodia 申请免费的API
  2. 打开 huggingface Prodia的hugging face的仓库
  3. 然后获取docker 部署命令


    docker 命令格式为
    docker run -it -p 7860:7860 --platform=linux/amd64 \ -e PRODIA_API_KEY=YOUR_KEY \ registry.hf.space/prodia-fast-stable-diffusion:latest python app.py

    把你获取的api_key填在YOUR_KEY处就可以然后去服务器运行就可以
